# aliasy_v1

Ten katalog przechowuje kopie dawnych modułów aliasowych z poziomu głównego
`microbe_radiation_model/`, które zostały usunięte po twardym cięciu struktury.

## Cel
- zachowanie historycznej referencji implementacji aliasów
- możliwość porównania poprzedniej i obecnej organizacji pakietu
- wsparcie ewentualnej ręcznej migracji starych importów

## Status
- pliki w tym katalogu nie są częścią aktywnego runtime
- nie są oficjalnym API
- nie powinny być importowane przez nowy kod
